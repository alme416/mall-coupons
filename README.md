# mall-coupons
- 给在商场购物的顾客提供的优惠券小程序
- 用户可以在商场的大屏幕或者其他海报上扫描二维码，打开小程序购买对应的优惠券
- 在支持优惠券的商家扫描二维码，打开小程序使用优惠券抵扣消费金额
